# 上海法院文书一键下载 
[![Manifest Version](https://img.shields.io/badge/Manifest-v3-blue)](https://developer.chrome.com/docs/extensions/mv3/intro/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

Chrome 浏览器扩展插件，用于自动批量下载上海法院诉讼服务网（http://www.hshfy.sh.cn）文书送达页面的法律文书。

##  功能特性

- **一键批量下载**：自动识别文书送达页面中的所有可下载文书
- **智能命名**：根据案号、文书类型自动重命名文件，方便归档管理
- **本地存储**：直接保存至浏览器默认下载文件夹，支持修改保存路径
- **安全合规**：仅操作页面可见的公开文书链接，不涉及数据抓取或接口破解

##  安装方法

1. 下载本项目源码并解压
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目文件夹
5. 安装成功，浏览器工具栏将出现插件图标


##  使用说明

1. **进入页面**：访问 [上海法院诉讼服务网](http://www.hshfy.sh.cn),导航至「文书送达」或「电子送达」页面
2. **启动插件**：点击浏览器工具栏中的插件图标，打开弹窗
3. **开始下载**：
   - 点击「一键下载全部文书」按钮
   - 插件将自动扫描当前页面中的所有 PDF/Word 文书链接
   - 自动批量下载至本地（默认下载文件夹：`~/Downloads/法院文书/`）
